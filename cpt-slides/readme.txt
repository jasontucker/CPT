== Description ==

Creates a Custom Post Type for Slides including custom columns, meta boxes and shortcode. 

== Installation ==

1. Upload the plugin to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click Slides in the left dashboard menu
4. Create a new Slide and click Publish
5. To add the shortcode, create a page and use [slides]
